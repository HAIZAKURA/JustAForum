# JustAForum
 A Demo Forum with SSM.

## Recommend

```
JDK 1.8+
Maven 3.6.2+
Tomcat 9.0+
MySQL 5.5+
IntelliJ IDEA
```

## SQL Structure

```
forumdb
  ├─ users
  |    ├─ uid
  |    ├─ uname
  |    ├─ upass
  |    ├─ umail
  |    ├─ uacce
  |    └─ upoit
  ├─ nodes
  |    ├─ nid
  |    ├─ nname
  |    └─ ncont
  ├─ tags
  |    ├─ gid
  |    ├─ gname
  |    └─ nid
  |
 ...
```
